# Hi, I'm Mike 👋

## My Story

I am not a developer by trade. I had never written a single line of Python before this project.

What I had was a problem to solve, a curiosity about AI, and enough stubbornness to figure it out one error message at a time.

I started with zero knowledge of Flask, APIs, WebSockets, databases, or machine learning. ---

## What I Built

**Personal Neuro Layer** is a fully working personal AI assistant I run in my own home. Not a demo. Not a tutorial project. A real system I use every single day.

It manages my:
- 🏠 Smart home via Home Assistant
- 📧 Email, documents, contacts
- 🔐 Network security with auto-banning
- 🎵 Music through Music Assistant
- 🌐 MikroTik router and QNAP NAS
- 💰 Cardano crypto trading with live signals and paper trading
- 🧠 Long-term memory with vector search
- 🎤 Wake word listener and voice interface
- 📱 Mobile face UI accessible via Cloudflare tunnel
- 🔍 Web search and live news
- ✍️ Self-writing code review system

The architecture — a central routing layer with specialized modules hanging off it — mirrors what the AI industry is now recognizing as the right way to build personal AI systems. I arrived at it independently by just asking *"how would a brain work?"*

## System Requirements

**LM Studio** is required to run the AI brain locally. Performance depends entirely on your hardware.

Recommended minimum specs:
- **RAM:** 16GB (32GB recommended)
- **GPU:** NVIDIA GPU with 8GB+ VRAM for best performance
- **CPU:** Modern 6-core or better (CPU-only mode works but is slow)
- **Storage:** 20GB+ free for models

The default model used is **Qwen2.5-14B** — a strong balance of intelligence and speed on mid-range hardware. If your machine struggles you can swap to a smaller 7B model in the config with no other changes required.

No cloud API required — everything runs locally on your machine.
